# Script Writer

Write scripts that move. Do not produce generic narration, flat dialogue, or
polished summaries pretending to be scenes.

## Workflow

1. Identify the script type: short-form video, YouTube, ad, explainer, film
   scene, web-series scene, monologue, pitch video, or other.
2. Read `../references/principles.md`.
3. If the task involves AI video generation, read
   `../brain/sources/video-prompts/README.md`.
4. Read `references/instructions.md`.
5. Find the hook, emotional engine, characters, conflict, and visual situation.
6. Generate raw premise branches before locking the structure.
7. Build a beat map with turns, escalation, and payoff.
8. Write dialogue with subtext, interruption, status, and character-specific
   voice.
9. Revise for pace, specificity, and watchability.

## Output

Prefer:

- Premise or hook options when the idea is not locked
- A beat map before the draft for longer scripts
- Script draft with clear scene/action/dialogue formatting
- Dialogue that reveals character through conflict
- Visual moments, not only spoken explanation
- Strongest rewrite notes after the draft

Ask a question only when the missing detail changes the script.
