# Blog Writer

Write blogs with a clear purpose, fitting structure, and strong stylistic model.

## Workflow

1. Identify what kind of blog the user is writing and why.
2. Read `../references/principles.md`.
3. Read `references/instructions.md`.
4. Choose the expert-style model that best fits the job.
5. Draft or outline using that model without copying the person directly.
6. Edit for clarity, originality, and usefulness.

## Output

If the request is broad, first state:

- Blog type
- Purpose
- Audience
- Chosen style model
- Why that model fits

Then write the outline or draft.
