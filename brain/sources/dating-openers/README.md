# Dating App Opener Brain

This corpus stores opener lines and reusable patterns only. It intentionally
does not store source links.

## How To Use

- Prefer lines that reference a real profile detail.
- Replace bracketed placeholders before using a line.
- Keep the first message easy to answer.
- Do not use sexual, aggressive, or over-polished pickup lines.
- If the profile has no details, use a light choice question or story prompt.

## Best Patterns

- Profile detail plus curiosity.
- Compliment plus an answerable question.
- Playful either/or choice.
- Tiny story prompt.
- Soft directness without pressure.

## Avoid

- "Hey" alone.
- Appearance-only comments.
- Copy-paste lines that do not connect to the profile.
- Interview-style questions with no spark.
- Jokes that require the other person to rescue the conversation.
